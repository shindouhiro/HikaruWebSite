| 配置项 | 说明 |
|--------|------|
| commands | 注册的命令 |
| keybindings | 键盘快捷键绑定 |
| menus | 菜单项(命令面板/编辑器上下文菜单等) |
| views | 侧边栏视图容器 |
| viewsContainers | 自定义视图容器 |
| configuration | 插件配置选项 |
| languages | 编程语言支持 |
| grammars | TextMate语法定义 |
| themes | 颜色主题 |
| iconThemes | 文件图标主题 |
| snippets | 代码片段 |
| debuggers | 调试器支持 |
| breakpoints | 断点类型 |
| taskDefinitions | 任务定义 |
