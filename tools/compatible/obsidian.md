# Obsidian 知识库管理工具

Obsidian 是一个基于 Markdown 文件的本地知识库管理工具。

## 核心特性

- **本地存储**: 所有数据保存在本地，安全可控
- **链式笔记**: 通过双向链接建立知识网络
- **插件生态**: 丰富的社区插件支持
- **多端同步**: 支持 iCloud、Git、Obsidian Sync 等多种同步方式

## 常用插件推荐

1. **Dataview**: 强大的数据查询语言
2. **Excalidraw**: 优秀的绘图工具
3. **Templater**: 强大的模板引擎
4. **Kanban**: 看板管理

## 为什么选择 Obsidian?

Obsidian 的核心优势在于其灵活性和可扩展性。你可以把它当作简单的 Markdown 编辑器，也可以通过插件将其打造成为个人的第二大脑。
